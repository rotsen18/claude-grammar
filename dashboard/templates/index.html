about grammar·log

A local Claude Code hook that grammar-checks every prompt you submit and streams the diffs to this dashboard. Nothing leaves your machine except API calls to the selected corrector.

correctors

groq Fast (~2s), structured JSON output. Primary model is llama-3.3-70b-versatile; on 429 rate-limit it falls back automatically to llama-4-scout-17b, then qwen3-32b. Requires an API key (free tier). → console.groq.com
claude_cli Highest quality. Invokes claude -p with Haiku in non-interactive mode — no API key needed, uses your existing Claude subscription. Slower (~5-10s) but best at tense/structure.
languagetool Free, rule-based grammar engine — no LLM. Good for spelling and typos, weak on tense/word-order/structure. ~20 req/min public limit. → languagetool.org

special markers

,, separator — text above it is natural language to correct; everything below is data/code and ignored.
^^^ bypass marker — ends the prompt with this to skip grammar correction entirely.
